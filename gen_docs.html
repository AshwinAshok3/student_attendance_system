SENTINEL AI

Offline Facial Recognition Attendance System

System Design Document (v2.0)

1. System Overview

Sentinel AI is a local, GPU-accelerated attendance system. It captures video feeds, identifies students using the InsightFace model, and marks attendance based on time-slot logic. The system operates in a completely air-gapped (offline) environment.
