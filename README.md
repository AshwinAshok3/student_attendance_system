## üöÄ Installation Guide (Step-by-Step)

### Step 1: Clone the Repository
Open your terminal (Command Prompt or PowerShell) and run:
```bash
git clone [https://github.com/YOUR_USERNAME/Sentinel-AI.git](https://github.com/YOUR_USERNAME/Sentinel-AI.git)
cd Sentinel-AI
```

### Step 2: Run the One-Click Installer
This script will download a standalone Python environment and all 3GB of GPU libraries into the project folder. It does NOT require admin privileges.

1. Right-click `install_local.ps1`.

2. Select **"Run with PowerShell"**.

3. *Alternative*: Open PowerShell in the folder and run:
```bash
powershell -ExecutionPolicy Bypass -File .\install_local.ps1
```
4. Wait until you see the message: `‚úÖ SETUP COMPLETE!` (This takes 5-10 minutes depending on internet speed).   


### 1. Add Student Data (Optional)
The system learns faces automatically.

1. Go to the `data/students` folder.

2. Create a folder for each student (e.g., `data/students/John_Doe`).

3. Paste 1 or 2 clear photos of their face inside that folder.


### 2. Launch the System
Run `.\start_app.bat` file from **powershell**.

* **First Run**: You will see a black window downloading the AI models (approx 300MB). **Do not close it**.

* Subsequent Runs: The system will launch instantly and open your browser to `http://localhost:8501`.


### Troubleshooting
Q: The black window closes immediately! 

You likely don't have an NVIDIA GPU or your drivers are old. Update your NVIDIA drivers.

Q: "Execution Policy" Error during install? A: Run this command in PowerShell first: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Q: Where is the data stored? A:

* Photos: `data/students/`

* Logs: `database/attendance.db` (SQLite)

* Encodings: `database/embeddings.pkl`
